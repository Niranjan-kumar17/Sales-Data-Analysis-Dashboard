# Sales-Data-Analysis-Dashboard

# Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI
- Excel
  

# Project Overview
Built an end-to-end Sales Data Analysis Dashboard using Python, SQL, and Power BI. The project includes data generation, cleaning, SQL analysis, and an interactive dashboard.

# Key Features
- Total Sales
- Total Profit
- Profit Margin
- Monthly Sales Trend
- Category-wise Sales
- Region-wise Sales
- Top 10 Customers
- Top 10 Products
- Payment Mode Analysis
- Interactive Slicers
